# 爬山演算法 -- 最簡單的優化算法



## 爬山演算法

```js
Algorithm HillClimbing(f, x)
  x = 隨意設定一個解
  while (x 有鄰居 x1 比 x 更高)
    x = x1
  end
  return x
end
```

# DFS(Depth-first Search)深度優先搜尋法

## 深度優先搜尋法是一種跑遍一個樹或圖的演算法。
* 開始的起點為一個樹的根(如圖)，或是一個圖的一個節點(如圖)，最先是搜尋與根相連接的節點且未被搜尋過，並且盡可能的往深處搜索，直到該節點的所有邊上都已被搜尋，接著，就回溯到前一個節點，重複做搜尋深處節點的動作，如果與該節點相連的所有節點都已循遍，就再往前回溯，直到整個樹或整個圖都被搜尋完畢。
深度搜尋法屬於一種盲目搜尋的方式，是利用堆疊(stack)的方式進行，通常以遞迴的方式呈現。

演算法：
```
  函數 DFS(頂點){
      標記 走過的節點為已走訪
      for每個相鄰的節點{
          如果沒有走訪過{
              DFS(參數)
          }
      }
  }
```

### 程式解說：
* DFS.js這個檔案用來實作這個概念，內容為走迷宮，地圖本身為固定的，設置開口在陣列 m[7,0]，然後依序判斷右邊、下面、左邊、上面，如果陣列內容為空白就替換為 'o'，如果為牆壁則跳出，然後不斷地走如果走到死路就離開遞迴，並把回溯的路設置為 'x'，最後便可以得知通往出口的路徑，一路將會都是 'o'，而顯示 'x' 則表示為死路。

程式參考老師版本：https://misavo.com/view/ai/search_dfs_mousepath.md

在某一行老師寫了 m[x][y]='+'; 這行程式碼，但後來發現 Javascript 不能對於陣列值直接做修改，所以這樣寫會被直接忽略，以至於老師的版本沒有 Show 出回溯的過程。
